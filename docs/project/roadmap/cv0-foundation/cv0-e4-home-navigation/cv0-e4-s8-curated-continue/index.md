# S8 — Continue surface becomes curated

The home's *Continue* band evolves from "active session + 3 most recent" into a curated surface where the mirror chooses what to surface and *why*. Replaces chronology with intention.

Possible surfacing reasons (to be refined when planned):

- **Recently active** — what you're working on right now
- **Open thread** — last message was yours; the mirror hasn't replied yet, OR the conversation ended on a question that didn't resolve
- **Returning theme** — you've come back to this topic N times in the last weeks
- **Dormant scope** — an organization or journey hasn't been visited in a while but has accumulated material
- **Mirror suggested** — reception or a similar layer noticed a thread worth picking back up

Each surfaced row carries the reason as a quiet label, not just a timestamp. The mirror as agent — not as inbox.

**Status:** Queued, plan deferred. Lands after S5 (scope ateliê) so we have real usage signal about what users actually return to.

**Why deferred:** the design space is wider than S5's. The reasons above are hypotheses; the actual mix should be informed by S5 usage. Writing the plan now would over-specify before we have data.
