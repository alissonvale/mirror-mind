# S8 — Cognitive Map

A `/map` surface that makes the mirror's structure visible and editable. The psyche's architecture — self (soul), ego, personas, skills — becomes a workspace of cards, one per layer. Empty layers (skills today, shadow and meta-self tomorrow) show invitations rather than blanks, signaling what the map can grow into.

**Structure, not memory.** The map holds who the mirror *is*. Memory — what the mirror *carries* — lives on other surfaces: the Context Rail for attention, a future episodic browse for accumulated conversations. See [2026-04-18 decision](../../../../decisions.md#2026-04-18--cognitive-map-and-memory-are-distinct-concepts-s8-renamed) for why the two are named separately.
