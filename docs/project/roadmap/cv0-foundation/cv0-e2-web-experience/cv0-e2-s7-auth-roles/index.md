# S7 — I know who's logged in

The sidebar makes identity and authority visible: avatar + name of the logged-in user, and a menu that adapts to role. Admin sees Users; regular user only sees Chat. Admin-only routes refuse non-admins with 403.

No dashboard, no profile page — a small, honest change that teaches the system two things: **who is here** and **what they are allowed to touch**.
