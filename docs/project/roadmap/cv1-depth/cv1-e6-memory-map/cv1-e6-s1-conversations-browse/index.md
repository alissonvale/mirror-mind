# S1 — Conversations browse

A new top-level surface at `/conversations` that lists every session the user has, with filters by persona, organization, and journey. Sorted by recency, paginated. The first cross-scope view of episodic memory.

The sidebar gains a **Conversations** (plural) entry alongside the existing **Conversation** (singular, continues to drop into the active session). One word's difference, two distinct purposes:

- **Conversation** → continue what I'm in
- **Conversations** → browse what I have

Anti-pattern to the chatbot sidebar in one essential way: filters are first-class. Sessions are not just chronological dump — they're discoverable across the structure (persona × org × journey) the rest of the mirror builds.
