# Test Guide: CV1.E1.S2 — Admin personas page (retroactive)

## Manual verification

### 1. Navigate to user profile

From Users page, click "Edit" on a user. Scroll to "Personas" section.

### 2. View personas

Expect: all user's personas listed as collapsible cards with name and content preview.

### 3. Edit a persona

Open a card, edit content, save. Expect: flash message confirms.

### 4. Delete a persona

Click "Delete" on a persona. Confirm. Expect: persona removed.

### 5. Add a persona

Open "+ Add persona" card. Enter id and content. Submit. Expect: new persona appears.
