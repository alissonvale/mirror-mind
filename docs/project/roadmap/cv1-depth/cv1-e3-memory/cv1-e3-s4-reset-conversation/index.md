# S4 — I can reset my conversation ✅

Two session-lifecycle actions, explicit and user-initiated, placed in the right periphery of the mirror alongside attention stats. **Begin again** creates a fresh session; the old one stays in the DB for future recall. **Forget this conversation** destroys entries and the session row — irrecoverable by design.
